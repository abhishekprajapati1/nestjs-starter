# NestJS + TypeScript + Swagger + Prisma Starter

## Description

This is a starter template for a NestJS application that uses TypeScript, Swagger, and Prisma. It provides a basic structure for building a RESTful API with a PostgreSQL database.

## Dependencies

- Redis Server

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
